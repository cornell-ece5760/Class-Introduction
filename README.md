# Setup-Guide
This is the setup guide for the ECE 5760 - FPGA Acceleration Course.

# Required Software
- Git/GitHub
- Vivado
- Vitis
- UVM
  - [UVM Guide by Chipverify](https://chipverify.com/uvm/uvm-installation)
- Cocotb (Could be a plus)
  - [Cocotb Documentation Guide](https://docs.cocotb.org/en/stable/#)

# Required Languages
Students should have a relatively concrete understanding of the following
programming and hardware description languages
- Verilog/SystemVerilog
- C/C++
- Python (Could be a plus)
